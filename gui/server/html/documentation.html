Chapter 1: Partners

Structure files

For each docking partner a separate PDB file specifying its residues, atoms and atomic coordinates has to be provided. The Protein Data Bank (PDB) format describes the three-dimensional structures of macromolecules derived from X-ray diffraction and NMR studies. You can find more information on the file format here and here. A typical PDB file contains lines like the following:
ATOM 1 N PRO A 1 8.316 21.206 21.530 1.00 17.44 N
ATOM 2 CA PRO A 1 7.608 20.729 20.336 1.00 17.44 C
ATOM 3 C PRO A 1 8.487 20.707 19.092 1.00 17.44 C
ATOM 4 O PRO A 1 9.466 21.457 19.005 1.00 17.44 O
ATOM 5 CB PRO A 1 6.460 21.723 20.211 1.00 22.26 C

RCSB PDB ID

Alternatively to providing PDB files for the protein partners, an RCSB PDB ID can be provided. This is a 4-letter code which identifies protein structures in the protein data bank (e.g. 3D5S). This option is not available for the ATTRACT easy interface.

If you do not want to use all chains provided by the PDB file for docking, you can specify here which one to use (e.g. if there is a dimer in the provided PDB file, you may want to use only chain A or chain B in the docking). This option is not available for the ATTRACT easy interface.

Chapter 2: Grids

Please note: using grids is VERY demanding in terms of memory. Do NOT use this option when your machine has little memory!
Calculating electrostatic and van der Waals interactions between the docking partners is computationally expensive. To significantly speed-up the docking process it is possible to use precalculated interaction grids in ATTRACT. The potential energy is calculated before the docking run and stored for each voxel. The actual potential energy for each atom is interpolated between eight voxels. Additionally, for each grid point a neighborlist is determined listing all the atoms which are within a certain range of this grid point. For all the atoms in the neighborlist the interactions will be calculated during the docking to calculate the correct short-range interactions. Typically the interactions in a range up to 10 A will be calculated directly and the potential energy value at the grid point will be corrected by this value.

In principle for each molecule a grid has to be calculated or provided. However, for two body docking only a receptor grid has to be calculated when fixing the receptor's center of mass (which is what you normally want!).

Chapter 9: Computation

ATTRACT uses a DOF (degrees of freedrom) file format to store protein complex structures. The protein complex structure can be decomposed into the structures of the individual proteins and their relative position and orientation to each other. For each structure the DOF file stores the degrees of freedom for each protein partner on a single line. It lists the rigid body degrees of freedom (three Euler angles for the orientation and the spatial position of the center of mass), then if applicable normal mode deformations and interface modes. ATTRACT internally applies these degrees of freedom to the given template structures (PDB files of the individual protein partners) to obtain the coordinates of the protein complex.
